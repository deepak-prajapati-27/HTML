TAG:

Tag are those which comes in between <angular bracket>

Example: <h1>

ELEMENT:

Start Tag - Some Content - End Tag.

Example: <p> MY NAME IS DEEPAK PRAJAPATI </p>

ATTRIBUTES:

They Provide Some Additional Information Which Change The behaviour of Element.
Always comes in Start Tag.

Example: <img src="../Images/Tagvselement.png" alt="Images" />

HEADING TAG:

Heading tag is used to give heading to a paragraph.
Only six heading are there i.e. h1 to h6.
h1 is the biggest and h6 is the smallest.

Example:

Deepak Prajapati - h1

Deepak Prajapati - h2

Deepak Prajapati - h3

Deepak Prajapati - h4

Deepak Prajapati - h5
Deepak Prajapati - h6

PARAGRAPH TAG:

<p> </p> is used to display paragraph.

Example: <p> I have started learning coding by Chetan Sir. </p>

hr TAG:

hr tag is used to give partion by drawing a line.
hr is a self closing tag.

Example: Deepak Prajapati <hr> Proud to be an Indian

Deepak Prajapati

Proud to be an Indian

br TAG:

br tag is used to break the line.
br is also self closing tag.

Example: <p>My self Deepak Prajapati. <br> I am from Umbergoan, gujarat.<br> I want to be a front end developer. </p>

My self Deepak Prajapati.
I am from Umbergoan, gujarat.
I want to be a front end developer.

pre TAG:

pre tag is used to display exactly the same as written.

Example: 

           My 
                                      is 
                                             Deepak   
                                                           Prajapati

BLOCKQUOTE TAG:

blockquote tag is used to give spacing to the content from both the side right and left of the screen.

Example: <blockquote>My name is Deepak Prajapati. I want to become a front end developer. I have done BCA from Rofel GM Bilakia college. My home town is at Rajasthan. I love to travel. And i love to go for a bike rides. </blockquote>

My name is Deepak Prajapati. I want to become a front end developer. I have done BCA from Rofel GM Bilakia college. My home town is at Rajasthan. I love to travel. And i love to go for a bike rides.

q TAG:

q tag is used to give double quotes to the content.

Example: <q>My name is Deepak Prajapati</q>

my name is Deepak Prajapati

marquee TAG:

Marquee tag is used for moving the content.
We can give direction from where to start left or right.

Example:
<marquee behavior="" direction= "left"> My name is Deepak Prajapati.</marquee>
<marquee behavior="" direction= "right"> My name is Deepak Prajapati.</marquee> My name is Deepak Prajapati. My name is Deepak Prajapati.
