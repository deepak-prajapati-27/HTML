WEBSITE:

Collection of web pages and media over the internet is called as website.
Website are accessible through a domain nsme on internet.
Website is hosted on a web server. We can access website using different web browser such as google chrome, firefox, brave, safari,etc.
Website are designed for various purpose such as education, e-commerce, entertainment, social networking and many more.
Every website has its own unique address known as URL(uniform resource relocater).

WWW:

WWW stands for 'world wide web'.
WWW is seen at the beginning of web address.
WWW is a subdomain that helps request to the right location on the internet.
Some website dont use WWW as the part of the address and stay with the domain.

HTTP/HTTPs:

HTTP stands for 'hypertext transfer protocal' and HTTPs stands for 'hypertext transfer protoccal secure'.
When we write a website name and hit enter our browser uses http to request the web page from the server an display it.
The information we send and recieve using http mostly can be viewed by others.
HTTP data is not encrypted.

HTTPs is more secure than HTTP.
The data exchanged between browser and web server is encrypted.
HTTPs is a extra layer of protection.
Most morden website and browser prefer HTTPs to protect data of user.

CLIENT SERVER:

The device that makes a request information or services from another device is know as client server.
When we use a web browser to access a website our device act as client.
The device which respond to the request made by client and provides the information and services is known as server.
Server is a powerful computer that stores and manage data.

DOMAIN:

Domain is the address of the website. Domain name is unique and easy to remember names to find the website.
Domain name is unique so two website cannot have the same domain name.
After domain domain name there is domain extension that indicates the organization the domain name is associated with it.
There are differne types of extension mostly common is ".com", and others are ".net",".org", etc.

HOST:

Host is a computer or devices which provide information to other devices over the internet.

HOSTING:

Hosting means the process of storing and making a website accessible on the internet.
Company provides hosting service that manage the servers.

FTP:

FTP stands for 'file transfer protocol'.
FTP is used to transfer files between devices over the internet.
It is a digital delivery services of files.
FTP is mostly used by web developers , designers to upload file to website server, so website can be easily accessed by visitor over internet.

PATH:

Path is like set of directions that tells the computer where to find specific file or folder on the computer or on the internet.

EXTENSION:

Extension is a small addition to the end of the filename to indicate the type of file.
It helps the computer to find out the type of file.
There are many type of extension such as .html, .txt, for images .jpg, .png, and many more.

EDITOR/ VSCODE:

Editor is a software tool that modify, create, format text and other contents on the computer.
Vscode is a code editor it is used by web developers to code the website and modify the website.

URL-URI/URN:

Url stands for uniform resource locator.
Url is the website address.

ADDRESS BAR:

Address bar is the the text box at the top of the web browser where we can type website address.

TITLE BAR:

Title bar is at the top of the window on the computer.
Title bar display the name of the program or file.
