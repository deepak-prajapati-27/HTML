<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<title>DEEPAK</title>
</head>
<body>
<h3> My Name Is Deepak Prajapati. </h3>
</body>
</html>

OUTPUT:

My Name Is Deepak Prajapati.

DOCTYPE Declaration (<!DOCTYPE html>):

This declaration defines the type of document being used. In this case, we're using the HTML5 doctype. It helps the browser understand which version of HTML is being used in the document. Using the correct doctype ensures that the browser renders the webpage in standards-compliant mode, preventing compatibility issues and ensuring consistent rendering across different browsers.

Meta Tags:

1. <meta charset="UTF-8">

This meta tag specifies the character encoding for the document. UTF-8 is a character encoding that supports a wide range of characters from different languages. It ensures that the text on the webpage is displayed correctly, regardless of the language used.
